# Meti0X7CB

Single-page personal site built with Next.js App Router + Tailwind, exported as a static site for GitHub Pages.

## Local development

```bash
pnpm install
pnpm dev
```

## Local production preview

```bash
pnpm build
pnpm start
```

## Deploy

Deployment is handled by `.github/workflows/deploy-pages.yml` on push to `main`.
